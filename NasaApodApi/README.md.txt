# My Nasa APOD App

This Android application displays NASA's Astronomy Picture of the Day (APOD) using the MVVM architectural pattern.

## Features
- Displays the title, image, and explanation of the APOD.
- Stores the last viewed APOD locally using Room.

## Prerequisites
- An API key from NASA's [APOD API](https://api.nasa.gov/).

## Getting Started
1. Clone this repository.
2. Replace `YOUR_API_KEY` in `Constants.kt` and `strings.xml` with your NASA API key.
3. Build and run the app in Android Studio.

## Technologies Used
- Kotlin
- MVVM Pattern
- Retrofit
- Room Database
- LiveData and ViewModel
- Picasso for image loading
